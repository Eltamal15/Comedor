Agregar

Para poder agregar un nuevo producto el cliente tiene que llenar los espacios de nombre, proveedor, descripcion, y cantidad obligatoriamente.

En el apartado de nombre el usuario tendra que ingresar el nombre, no podra ingresar una cadena menor a 3 digitos ni una mayor a 30.

El proveedor se desplegara una lista con los proveedores ingresados al sistema, en caso de que no aparezca ni un proveedor tendra que agregar uno ya que podria ejecutar un error en el programa.

La descripcion del producto un espacion obligatorio llenarlo.

El campo de observaciones no es un campo obligatorio de llenar pero es recomendable en todo caso que necesite seguir instrucciones ej.(Mantenerse refrigerado).

En el campo de cantidad la primera vez se necesita ingresar la cantidad se va registrar, esta se modificara automaticamente siempre que se realice un platillo con cierta cantidad. Para sumar mas cantidades del ingrediente es necesario apretar el boton de suma que esta al lado. (+)

Una vez que el usuario haya verificado que los datos que va ingresar son correctos solamente necesitara apretar le boton "Agregar" y se hara la creacion del producto y se vera reflejado en la tabla.
Tomar en cuenta que si se introduce un producto que ya existe mostrara un mensaje de error.

Modificar

Para modificar un producto el usuario tendra que seleccionar el producto de la tabla, una vez ya seleccionado se mostraran automaticamente en sus campos el usuario ya podra modificar el campo necesario.
Tomar en cuenta que si modifica por un producto existente mostrara un error.

Sumar cantidad

Para poder sumar la cantidad el usuario necesita ingresar la cantidad de producto nuevo en el espacio de "cantidad a sumar" seguido de eso apretar el boton de suma para que se realice automaticamente.

Reporte

Para que el usuario pueda ver el reporte de inventario tendra que hacer clik sobre el boton de reporte inventario.
Una vez seleccionado se ejecutara el reporte para poder imprimirlo.
