Agregar

Para poder agregar el usuario neesita primero ingresar el DNI el cual tiene que ser de 13 digitos y estar en un rango de a√±o entre 1950 y 2006.

En el apartado de nombre el usuario no podra ingresar una cadena menor de 3 digtos, asi mismo tampoco podra ingresar una cadena de 30 caraceteres. El nombre tampoco podra contener dos letras seguidas.

En el apartado de campo el programa viene con 3 cargos predefinidos los cuales son: Cocinero, Limpieza, Seguridad. El usuario tendra que seleccionar uno.

La fecha no se podra modificar ya que es el dia en que se esta agregando el empleado en la planilla.

En el apartado de apellido el usuario no podra ingresar una cadena menor de 3 digtos, asi mismo tampoco podra ingresar una cadena de 30 caraceteres. El nombre tampoco podra contener dos letras seguidas.

El telefono fijo del empleado tendra que comenzar en 2 y solo podra agregar 8 digitos.

El celular del empleado los primeros valores podran ser 3, 8 y 9.

El celular de emergencia servira como respaldo. Es necesario que se ingrese.

El estado civil el usuario tendra dos opciones las cuales sera soltero y casado.

Una vez con los datos completos y verificados por el usuario que esten correctos tendra que presionar el boton de agregar y el usuario se creara automaticamente y saldra reflejado en la tabla.

Modificar

Para poder modificar un empleado el tendra que seleccionar el usuario en la tabla que tiene, una vez seleccionado el usuario, los datos del mismo se cargaran automaticamente en los espacios, el usuario tendra que modificar lo que desee y previamente tendra que apretar el boton de modificar para que se puedan guardar los cambios.

Pago

Para poder realizar el pago de un empleado el usuario tendra que seleccionar el empleado y seguido de eso se le desplegara la pantalla de pagos del empleado.

Agregar Pago

Para poder realizar el pago el usuario tendra que ingresar la cantidad de pago por dia, tambien esta el apartado de extras en el cual si no se realiza pagos extras se tendra que poner 0, la fecha se pondra automaticamente. Una vez el usuario haya verificado los datos tendra que apretar el boton de agregar y se realizara el pago y se mostrara en la tabla automaticamente.
