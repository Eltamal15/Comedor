Registrar Platillos

Para poder registrar un platillo el usuario necesitara ingresar el nombre del platillo, la descripcion y el precio del platillo, una vez verificado los datos el usuario tendra que apretar el boton "Agregar" y se mostrara el platillo en la tabla automaticamente.

Modificar

Para modificar el platillo el usuario tendra que seleccionar el platillo en la tabla que se muestra, una vez que se selecciona los datos del platillo se rellenaran automaticamente en los campos. Luego el usuario tendra que modificar los datos y para que se guarden tendra que apretar el boton "Realizar"

Ingredientes

El usuario para poder ingresar los ingredientes para cada platillo tendra que seleccionar el platillo y luego apretar el boton "Ingredientes" una vez seleccionado se le abrira una ventana nueva en la que se puede ver y registrar los ingredientes nuevos.

Una vez abierto la ventana de modificar ingredientes se tendra que seleccionar en la lista desplegable el ingrediente a utlizar y la cantidad, luego tendra que apretar el boton "Agregar".

La cantidad utilizada se se eliminara del inventario automaticamente.

Modificar ingrediente

A la hora de modificar el usuario solo podra modificar la cantidad del ingrediente utilizado, tendra que seleccionar el ingrediente a utilizar y en el campo de cantidad cambiar la cantidad. Una vez realizados los cambios tendra que apretar el boton de "modificar" y se guardaran los cambios automaticamente.
