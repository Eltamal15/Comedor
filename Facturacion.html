Crear factura

Para poder crear una nueva factura el usuario tendra que ingresar el nombre del cliente, una vez se haya ingresado el nombre del cliente tendra que abretar el boton de "Crear factura", luego se abrira una factura para poder seleccionar los platillos a llevar.

Datos de factura

Una vez el usuario haya creado la factura tendra que ingresar los datos de la factura los cuales son los platillos que llevara. El usuario tendra que seleccionar el plato y automaticamente se llenara el campo con la cantidad que esta disponible, el usaurio tendra que seleccionar la cantidad a llevar del cliente por platillo y posteriormente tendra que apretar el boton de "Agregar a pedido". Una vez el cliente complete los datos del pedido tendra que apretar el boton "Imprimir factura" para que se genere la factura y poder imprimirla.
