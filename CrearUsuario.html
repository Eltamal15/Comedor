Agregar

El usuario para poder crear un nuevo usuario tendra que ingresar el nombre de usuario, tendra que ingresar una contraseña y verificar que sea la misma, tambien tendra que ingresar un nombre de mascota que recuerde porque en caso que se olvide la contraseña este servira como respaldo, luego tendra que elegir el rol que tendra este en el sistema, cuando el usuario verifique que todos los datos estan correctos tendra que apretar el boton que dice "Crear Usuario" y se creara el usuario y estara listo para poder usuarse.
