Agregar

Para agregar un nuevo proveedor el usuario tiene que llenar obligatoriamente los campos de nombre, Direccion, Ciudad, Telefono.

Para poder registrar el nombre del proveedor el usuario tiene que ingresar una cadena mayor a 3 digitos y menor a 30.

En el campo de telefono el usuaro solo podra ingresar numeros que empiecen con 2 y solamente se aceptan 8 digitos.

En la direccion del proveedor tendra que ingresar donde esta ubicado el proveedor.

Una vez el usuario complete los datos del proveedor tiene que apretar el boton de agregar y se mostrara automaticamente en la tabla.

Modificar

Para poder modificar el usuario tendra que selecconar el proveedor que desee modificar y se completaran automaticamente los espacios, el usuario tendra que modificar los datos que desee y una vez verificados apretara el boton de modificar para que se guarden los datos.

Reporte

Para poder visualizar el reporte el usuario tendra que presionar el boton de "Reporte proveedores", una vez seleccionado se abrira la ventana para poder imprimir el reporte.
